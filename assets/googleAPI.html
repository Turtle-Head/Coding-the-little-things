Getting started with Google Maps

It is super easy to get started with Google Maps. The first thing to do is go over to Google Maps Javascript API and get a Key. You'll need the Key to do place searches and a few other things so it's best to have it handy before starting.
To create a Google Maps data request you'll need jQuery for it's AJAX import method. The best way to get it is by using a CDN to keep the file up to date. You can find one at code.jQuery.com.
In your html file create a map div.

<div id="map"></map>

We need to access the Google Map server with their script. We also need to include the API key to tell Google who is asking for the data.
<script async defer src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap"> </script>
Next, in your javascript, you need a few things.

function initMap() { // Create a map object and specify the DOM element for display. var map = new google.maps.Map(document.getElementById('map'), { center: {lat: 34.397, lng: -115.644}, zoom: 6 }); }


This gets you a basic map, centered over the longitude and latitude location provided.
